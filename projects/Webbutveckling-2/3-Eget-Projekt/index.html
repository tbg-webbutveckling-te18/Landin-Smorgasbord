Quick description of JavaScript

JavaScript is the most popular programming language in the world. It is one out of three languages that every webdeveloper needs to learn, among HTML and CSS. JavaScript is free to use for everyone. It has support for event-driven, such as click of a button, functional, and imperative programming styles. JavaScript allows you to make webpages interactive and engage the user with those interactive elements. The browser creates a Document Object Model(DOM) when the web page is loaded. It acts like an interface between the document and JavaScript. With this you can:

  • add, change and remove a HTML element or attribute
  • change any CSS styles
  • react to the existing events
  • create new events

The reason to why JavaScript has gotten so popular is because it is pretty easy to get started with. You don't need a environemnt setup, you can code in for example chromes Developer tool. JavaScript can run everywhere. It is a universal language and can run on laptops, mobiles, tablets and both client- and server-side. JavaScript has open standards and the two most popular engines, SpiderMonkey and V8, are open source projects. There are a lot of modern frameworks in JavaScript which are very developer friendly and have a good community for support.
